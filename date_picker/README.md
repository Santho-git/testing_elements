## The goal of this project is to automate testing a date picker
### Manaual test cases for testing a date picker
1. Verify date picker opens when clicked on the date field
2. Verify the date picker closes when a date is selected
3. Verify the date picker closes when clicking outside the calendar
4. Verify the date picker closes when esc key is pressed
5. Verify the selected date appears on the input field correctly after selecting the date
6. Verify the current date is getting highlighted
7. Verify next month navigation
8. Verify previous month navigation
9. Verify next year navigation
10. Verify previous year navigation
11. Verify selecting valid date
12. Verify date selection for different months and different years
13. Verify the first day of the month
14. Verify the last date of the month
15. Verify selection of february 28 during non leap years
16. Verify selection of february 29 during leap years
17. Verify valid entry of inputs in input fields
18. Verify invalid entry of inputs in input fields
19. Verify date selection using enter
20. verify date selction using arrow keys
21. Verify date selection using page up and page down
